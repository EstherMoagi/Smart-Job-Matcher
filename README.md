# Smart Job Matcher

##  Overview
Smart Job Matcher is a web application that helps users identify suitable job
opportunities based on their skills and preferences. The system matches user
inputs with relevant job roles  using cv the user uploaded.

## Features
- User-friendly web interface
- Job matching based on skills
- Backend logic implemented with Flask
- Dynamic content rendering

## Technologies Used
- Python
- Flask
- HTML
- CSS
